# Wisdom IT Readiness Toolkit v0.1.0

The first public release introduces a governed Technology Readiness Quick Check for individuals, founders, working professionals, and small businesses.

## Included

- Four-domain readiness assessment
- Weighted percentage scoring
- Five-level maturity classification
- Command-line scoring and Markdown reporting
- Synthetic demonstration data
- Unit tests and continuous integration
- Security, contribution, privacy, and licensing guidance

## Validation before tagging

- Confirm the Toolkit CI workflow passes on `main`.
- Download and inspect the generated example-readiness-report artifact.
- Run the Quick Check locally against synthetic responses.
- Confirm no real client data, credentials, or private records are present.

## Known limitations

- The CLI and Markdown report are the primary interfaces.
- The assessment is an advisory aid, not a regulatory certification.
- A browser-based accessible interface is planned for a later release.
