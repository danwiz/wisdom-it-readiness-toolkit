# Security Policy

## Supported version

Security fixes are applied to the latest release.

## Reporting a vulnerability

Do not open a public issue containing sensitive vulnerability details. Send a concise report to `danewisdom@gmail.com` with reproduction steps, affected files, and potential impact.

## Data handling

The toolkit operates locally and does not transmit assessment responses. Do not commit real client responses, credentials, secrets, or regulated information to this repository.
