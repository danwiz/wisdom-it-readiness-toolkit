# Wisdom IT Readiness Toolkit v0.1.0

## Overview

The first public release of the Wisdom IT Readiness Toolkit provides a governed, deterministic Technology Readiness Quick Check for individuals and small organizations. It includes a machine-readable assessment, weighted scoring, maturity classification, an installable command-line interface, Markdown report generation, tests, CI validation, and release evidence.

## Included in this release

- Technology Readiness Quick Check assessment (`WITRT-QUICK-001`)
- Four assessment domains: Strategy and Goals, Devices and Infrastructure, Cybersecurity and Resilience, and Support and Governance
- Weighted deterministic scoring with five maturity levels
- Installable Python package for Python 3.11 and later
- `witrt score` command for JSON scoring output
- `witrt report` command for Markdown readiness reports
- Synthetic demonstration responses and example report
- Unit tests for scoring and report generation
- GitHub Actions CI for package installation, tests, CLI smoke tests, report validation, and artifact upload
- MIT license, contribution guidance, security policy, architecture documentation, and changelog

## Installation

```bash
python -m pip install .
```

## Usage

```bash
witrt score assessments/quick-check-v0.1.json examples/synthetic-responses.json

witrt report assessments/quick-check-v0.1.json examples/synthetic-responses.json \
  --subject "Synthetic Demonstration" \
  --data-note "Synthetic demonstration only" \
  --output build/example-report.md
```

## Governance and limitations

- This toolkit supports structured readiness discovery and prioritization; it is not a certification, audit opinion, or guarantee of security or operational performance.
- Results depend on the accuracy and completeness of supplied responses.
- Material decisions should include human review and supporting evidence.
- Sensitive client or organizational data should not be committed to the public repository.
